Introduction - CSS

Ce cours s'adresse aux personnes ayant des notions voire maîtrisant le HTML. Si ce n'est pas le cas, je vous invite à suivre le cours sur cette technologie avant de passer au CSS. Le CSS (Cascading Style Sheets) complète le HTML, il s'occupe de la mise en forme d'une page web. Vous avez pu le constater lors de vos premiers pas, seul, le HTML est brut et sans saveurs. Grâce au CSS, nous allons pouvoir choisir la couleur du texte, la position des différents blocs et gérer l'affichage suivant la taille des écrans. Tout comme le HTML, le CSS a évolué au fil du temps et est aujourd'hui en version 3. On parle donc de CSS3. C'est également le W3C qui gère, encadre et définit ce langage. Vous naviguez sûrement tous les jours sur internet et vous vous êtes forcément aperçu que chaque site avait sa propre mise en forme. Cette dernière est indiquée au navigateur web par le lien entre le fichier HTML et la feuille de style CSS. Ce langage s'écrit dans un fichier indépendant qui se termine par l'extension .css.
