Chapitre 1 - Les sélecteurs et propriétés

Les sélecteurs CSS permettent de cibler des éléments HTML spécifiques afin d’appliquer des styles. Voici les principaux types de sélecteurs et quelques propriétés de base :

1.1 Les sélecteurs de base

- **Sélecteur universel** (`*`): applique des styles à tous les éléments.
- **Sélecteur de type** (ex : `p`): cible tous les éléments du type spécifié, comme tous les paragraphes. 

        p {
            color: blue; /* Les paragraphes seront bleus */
            font-size: 18px; /* La taille de 18 pixels me semble pas mal */
        }
- **Sélecteur de classe** (`.classe`): cible les éléments ayant une classe spécifique. 
        h1 {
            /* Mettez les propriétés CSS ici pour le titre h1 */
        }
        
        p {
            /* Mettez les propriétés CSS ici pour les paragraphes */
        }
- **Sélecteur d'identifiant** (`#id`): cible un élément unique ayant l'identifiant spécifié. 
        #logo {
            /* Mettez les propriétés CSS ici */
        }

1.2 Les propriétés de base

- **Couleur** (`color`): définit la couleur du texte.
- **Fond** (`background`): applique une couleur ou une image de fond.
- **Marge** (`margin`): espace extérieur autour d'un élément.
- **Padding**: espace intérieur entre le contenu et la bordure d'un élément.
