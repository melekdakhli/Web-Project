Introduction - CSS

Vous naviguez sûrement tous les jours sur internet et vous vous êtes forcément aperçu que chaque site avait sa propre mise en forme. Cette dernière est indiquée au navigateur web par le lien entre le fichier HTML et la feuille de style CSS. Ce langage s'écrit dans un fichier indépendant qui se termine par l'extension .css.

Le CSS (Cascading Style Sheets) complète le HTML et s'occupe de la mise en forme d'une page web...

Chapitre 1 - Les sélecteurs et propriétés

Ce chapitre couvre les différents sélecteurs CSS qui permettent de cibler des éléments HTML pour appliquer des styles. Il aborde également les propriétés CSS de base, comme la couleur, la taille de police, les marges, et bien plus.

Chapitre 2 - Héritage en cascade

La cascade et l'héritage sont des concepts importants en CSS, permettant de gérer les priorités entre les styles. Ce chapitre explique comment les règles CSS se chevauchent et comment le navigateur décide quels styles appliquer.

Chapitre 3 - Propriété display

La propriété display contrôle l'affichage des éléments HTML. Ce chapitre explore les valeurs de display comme block, inline, et flex, pour organiser et positionner les éléments sur la page.

Chapitre 4 - Le modèle de boîte

Le modèle de boîte est fondamental pour comprendre la mise en page en CSS. Il décrit comment la largeur, la hauteur, les marges, les bordures, et le padding d'un élément sont calculés.

Chapitre 5 - Les unités

Les unités permettent de définir les dimensions en CSS. Ce chapitre examine les unités absolues (px, cm) et relatives (%, em, rem), et leur utilisation.

Chapitre 6 - Typographie

Ce chapitre explore les propriétés de typographie en CSS, comme la police, la taille de police, la graisse, et les interlignes, pour améliorer la lisibilité du texte.
